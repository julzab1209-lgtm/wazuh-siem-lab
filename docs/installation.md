# Wazuh Installation

## Overview

The purpose of this project was to deploy a functional Wazuh SIEM environment capable of collecting and analyzing security events from Linux endpoints.

---

## Environment

- Oracle VirtualBox
- Ubuntu Server
- Kali Linux
- Wazuh Server
- Wazuh Dashboard

---

## Installation Steps

### 1. Virtual machine preparation

- Installed Ubuntu Server inside Oracle VirtualBox.
- Updated the operating system packages.

### 2. Wazuh deployment

- Installed Wazuh Server.
- Installed Wazuh Indexer.
- Installed Wazuh Dashboard.
- Verified that all services were running correctly.

### 3. Dashboard access

- Accessed the Wazuh Dashboard through a web browser.
- Verified successful authentication.

### 4. Endpoint preparation

- Prepared Kali Linux as the monitored endpoint.

---

## Result

A fully operational Wazuh SIEM environment was successfully deployed and ready for security monitoring.
